# dotfiles
Hello! Theese are my dotfiles, so you can configure everything like I do!

### Installation procedure

* nvim configuration
  * install nvim ([Nvim site](https://neovim.io/))
  * move files in the nvim configuration folder (probably .config/nvim)
  * run PackerSync to make sure all packages are installed and updated

* fish configuration
  * install fish (brew install fish)
  * move files in the fish configuration folder (probably .config/fish)
 
* git configuration
  * move file into root dir
  
* tmux configuration
  * install tmux
  * move file into root dir
